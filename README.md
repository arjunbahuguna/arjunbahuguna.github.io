#### My Blog

Backend for my Hugo blog using the [PaperMod](https://github.com/adityatelange/hugo-PaperMod/tree/exampleSite) theme by [Aditya Telange](https://github.com/adityatelange).

#### Quick start
- Run local server: `hugo server`
- Edit site config: `hugo.yaml`
- Override styles: `assets/css/extended/custom.css` (or edit Markdown in `themes/PaperMod/layouts/partials/templates`)
